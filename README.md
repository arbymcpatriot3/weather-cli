# Clean Shot v3.0
**Built for the road, not the boardroom.**

Weather and road intelligence for truck drivers.
Full-featured on a phone screen. Fast on 2G.

---

## What's in v3.0

- Weather + 7 road alert detectors (black ice, bridge freeze, fog, flood, diesel gel, high wind, mudslide)
- Community hazard reports with GPS geo-confirmation
- DOT/511 feeds all 50 states
- Smart parking runway (HOS-aware)
- HOS guardian — FMCSA 11/14/70 hour advisory tracking
- Text-to-speech voice alerts (CB-style)
- Flash + beep critical alerts, 5 severity levels
- GPS navigation math
- English + Spanish
- Referral engine
- Auto-updates
- One-line install on all 5 platforms

---

## Install

### Linux
```bash
curl -fsSL https://raw.githubusercontent.com/arbymcpatriot3/weather-cli/main/clean-shot/platforms/linux/install.sh | bash
```

### macOS
```bash
curl -fsSL https://raw.githubusercontent.com/arbymcpatriot3/weather-cli/main/clean-shot/platforms/macos/install.sh | bash
```

### Android / Termux
```bash
curl -fsSL https://raw.githubusercontent.com/arbymcpatriot3/weather-cli/main/clean-shot/platforms/android/install_termux.sh | bash
```

### Windows (PowerShell)
```powershell
iwr https://raw.githubusercontent.com/arbymcpatriot3/weather-cli/main/clean-shot/platforms/windows/install.ps1 | iex
```

### iOS / iSH
```sh
wget -qO- https://raw.githubusercontent.com/arbymcpatriot3/weather-cli/main/clean-shot/platforms/ios/install_ish.sh | sh
```
