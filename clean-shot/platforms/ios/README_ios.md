# Clean Shot — iOS Setup

**Built for the road, not the boardroom.**

## Requirements

- iPhone or iPad running iOS 15+
- [Pythonista 3](https://omz-software.com/pythonista/) ($9.99) or [Pyto](https://pyto.app/) (free)

## Install

1. Open Pythonista or Pyto on your device
2. Download the `clean-shot` folder to your app's Documents
3. Run `platforms/ios/main.py`

## Features available on iOS

- Full weather display
- NOAA alerts
- Route weather (5 stops)
- TTS via AVSpeechSynthesizer (built-in, offline)
- GPS via CoreLocation

## Limitations vs Linux/Windows

- Voice wake-word ("Hey Clean Shot") not yet available on iOS
- Background refresh requires the app to be open
